How to request data deletion for Cookieclicker2.mp4 (Raidenxd2)

There are 3 ways to request data deletion


1: In-game

Open Cookieclicker2.mp4 and go to Options > Save Management > Cloud Save... > Delete save data from the cloud

Confirm that you want to delete your data.

If this fails, see the 2nd way.


2: Contacting Raidenxd2

You should only need to do this if the first way failed.

Open Cookieclicker2.mp4 and follow the steps.

First, get your Player ID by going to Options > Save Management > Cloud Save... > Get Player ID

Second, send an email with the Player ID to raidenxd2@gmail.com

Third, disable Cloud Save by going to Options > Save Management > Cloud Save... > Disable Cloud Save

It may take up to 1 weak to have your data deleted. Once your data is deleted, you will recieve a reply to the email saying that your data has been deleted


3: Contacting Unity to request data deletion

You will need to send an email to DPO@unity3d.com

Unity Game Player and App User Privacy Policy

For more information, visit https://www.datarequests.org/company/unity3d/

Comment demander la suppression des données pour Cookieclicker2.mp4 (Raidenxd2)

Il y a trois façons de demander la suppression des données


1 : Dans le jeu

Ouvrez Cookieclicker2.mp4 et allez dans Options > Gestion des sauvegardes > Sauvegarde dans le nuage... > Supprimer les données d'enregistrement du nuage

Confirmez que vous souhaitez supprimer vos données.

En cas d'échec, voir la deuxième méthode.


2 : Contacter Raidenxd2

Cette opération n'est nécessaire que si la première méthode a échoué.

La suppression de vos données peut prendre jusqu'à 1 semaine. Une fois vos données supprimées, vous recevrez une réponse à l'e-mail indiquant que vos données ont été supprimées.


3 : Contacter Unity pour demander la suppression des données

Vous devez envoyer un courriel à DPO@unity3d.com

Cookieclicker2.mp4のデータ削除依頼方法（Raidenxd2）

データの削除を要求するには3つの方法があります。


1: ゲーム内

Cookieclicker2.mp4を開き、オプション＞保存管理＞クラウドセーブ...と進みます。> クラウドから保存データを削除する

データを削除することを確認します。

失敗した場合は、2番目の方法を参照。


2："Raidenxd2 "への問い合わせ

最初の方法が失敗した場合のみ、この方法を取る必要がある。

Cookieclicker2.mp4を開き、手順に従ってください。

まず、オプション＞セーブ管理＞クラウドセーブ...でプレイヤーIDを取得します。> プレーヤーIDを取得

次に、プレーヤーIDを記載したメールを raidenxd2@gmail.com に送信する。

第三に、オプション > 保存管理 > クラウドセーブ...でクラウドセーブを無効にします。> クラウドセーブを無効にする

データの削除には最大1週間程度かかる場合があります。データが削除されると、データが削除された旨のメールが返信されます。


3: Unityへのデータ削除の依頼

DPO@unity3d.com までメールをお送りください

Unity Game Player and App User Privacy Policy

詳細はhttps://www.datarequests.org/company/unity3d/。

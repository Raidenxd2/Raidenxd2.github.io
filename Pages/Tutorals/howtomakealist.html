How to make a list

First. Open up notepad.

2. Save as index.html IT MUST HAVE .html AT THE END!!! ALSO YOU CAN MAKE IT WHAT EVER YOU WANT...

3. Type this code:

Ordered list:

< !DOCTYPE html>
< head>
< /head>
< body>

< ol>
< li>Ordered list< /li>
< li>Ordered list< /li>
< li>Ordered list< /li>
< /ol>
< /body>

Without spaces in the tags



Unordered list:

< !DOCTYPE html>
< head>
< /head>
< body>

< ul>
< li>Unordered list< /li>
< li>Unordered list< /li>
< li>Unordered list< /li>
< /ul>
< /body>

Without spaces in the tags
